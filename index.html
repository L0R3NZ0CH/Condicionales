Condicionales

En JavaScript, las condiciones se utilizan para tomar decisiones en el código. Se basan en la evaluación de expresiones booleanas, que pueden ser verdaderas o falsas. Las estructuras de control de flujo, como if, else if, else, switch, permiten ejecutar diferentes bloques de código según el resultado de estas expresiones.

En este caso, el código verifica si la variable numero es mayor que 0, menor que 0 o igual a 0, y ejecuta el bloque de código correspondiente.

En este caso, la expresión "estaciones"se compara con diferentes casos en el switch. Si estaciones es "primavera", se ejecutará el código bajo case "primavera", y así sucesivamente. Si no coincide con ninguno de los casos, se ejecutará el bloque de código bajo default. Es importante recordar incluir break después de cada bloque de código para salir del switch una vez que se ha ejecutado el código correspondiente.
